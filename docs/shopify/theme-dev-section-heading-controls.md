# Section heading controls

## Summary
Section-level headings now support alignment (left/center/right) and size controls where applicable. These settings are available in the section editor for any section that exposes a heading field, defaulting to center alignment.

## Sections updated
- Background video, Blend page, Blends profiles, Blog posts, Brands, Collage, Collection feature, Collection list, Featured collection, Featured navigation, Featured product, Features banner, Image banner, Lookbook, Main 404, Main collection banner, Main list collections, Multicolumn, PB Hero - Topo, Product recommendations, Separator, Shoppable image, Shoppable video, Slideshow, Testimonials, Video reels, Video section.

## Editor guidance
- Heading alignment: use the alignment select to set left/center/right.
- Heading size: use the desktop/mobile scale sliders to adjust headline size.
- PB Hero - Topo: content alignment now supports left/center/right and the heading has desktop/mobile scale controls.
- Main 404: use the existing font size controls for mobile/desktop to adjust the headline size.

## QA checklist
- `/pages/blends`: "Our Blends" is centered by default and updates when alignment changes.
- Home page: verify any section heading alignment and size updates as expected.
